ORACLE is a work-in-progress experimental web experience that explores memory as something spatial, imperfect, and alive. Memories exist as objects in an interactive 3D orbit, containing fragments such as dates, times, places, music, images, and personal recollections. The interface combines 3D graphics, archival design, and a grayscale cyberpunk operating system aesthetic. ORACLE acts as a machine attempting to reconstruct and organize human memory.

## Built With

- React
- Three.js / React Three Fiber
- JavaScript
- CSS
